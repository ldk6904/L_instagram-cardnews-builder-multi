🔑 OpenAI API 키 설정

AI 리서치 기반 카드뉴스 생성을 위해 OpenAI API 키가 필요합니다. 키는 이 브라우저(LocalStorage)에만 저장되며 서버로 전송되지 않습니다.

현재 저장된 키: 미설정

📌 키 발급 방법

  • platform.openai.com/api-keys 접속 → 로그인
  • 우상단 Create new secret key 클릭 → 이름 입력 → 생성
  • 표시되는 키(sk-로 시작) 복사 → 이 화면에 붙여넣기
  • OpenAI 결제 수단 등록 필요 (결제 설정) · 최소 $5 충전 권장

💰 예상 비용

  • 모델: gpt-4o-mini ($0.15 / 1M input, $0.60 / 1M output)
  • 카드뉴스 1세트(8장) ≈ 약 $0.002 (3원)
  • $5 충전 = 약 2,500세트 생성 가능

📖 카드뉴스 빌더 사용법

5분이면 인스타용 카드뉴스 8장이 완성됩니다.

1

시작하는 3가지 방법

✨ 주제로 시작 — 주제 한 줄만 입력하면 8장이 자동으로 채워져요. 형식(팁/단계/비교) + 톤 + 카테고리 선택. 가장 빠른 길.
💡 예시로 시작 — 6개 완성본 중 하나 골라 시작. 톤 학습용으로 좋아요.
📝 직접 입력 — 처음부터 본인 손으로. 익숙해진 다음 추천.

2

좌측 폼 5단계 입력

STEP 1 주제 · STEP 2 레이아웃 · STEP 3 컬러 · STEP 4 본문 6장 · STEP 5 본인 정보. 입력하는 즉시 우측 미리보기에 반영됩니다.

3

카드 미리보기로 8장 확인

우측 미리보기 영역의 ‹ › 버튼이나 아래 썸네일 클릭으로 카드 1~8장을 모두 확인. 키보드 ← → 도 가능.

4

PNG 다운로드 → 인스타 업로드

우측 상단 ⬇ PNG 8장 ZIP 다운로드 버튼 → 압축파일이 받아짐 → 풀어서 인스타그램에 순서대로 업로드(캐러셀)!

💡 더 잘 만드는 팁

  • 표지(1번)는 후킹이 전부. "검색이 사라지는 시대"처럼 호기심을 자극하는 문장으로.
  • 본문 6장은 한 카드 한 메시지. 줄바꿈은 그냥 엔터.
  • 컬러는 본인 브랜드 컬러로. 프리셋이 마음에 안 들면 HEX 코드로 직접 입력.
  • 레이아웃은 한 시리즈에 통일. classic은 정보 위주, bold는 임팩트, minimal은 감성적인 주제에.
